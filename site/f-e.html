Lecture 8: Character Entity References
(Note: For some reason, I don't need to add a semi-colon for some Entities to show, that why I have enclosed the words in brackets)

There are 3 characters that should always be escaped. They are (<), (>) and (&).
To show these as text you must type in &lt, &gt, &amp respectively but they must be followed by a semi-colon.

Other example of Entity References:
(©): &[copy] ("): &[quot]

You can also stop text wrapping when a window is made smaller by using &[nbsp]. I've typed a long sentence so that I can show how to prevent broken words. The set of unbroken words that I am going to use as an example is my full name. Tiffany Oluwatoyosi Taiwo Jacobs.
